## 📋 About This Project

This is a professional portfolio website for **Bankesola Banjo** — a **Medical Virtual Assistant** specializing in healthcare administrative support, patient coordination, and remote practice solutions.

The site showcases her experience, certifications, services, and client testimonials with a premium dark and gold theme that reflects professionalism and reliability in the healthcare industry.

### ✨ Key Features

- 🏥 **Healthcare Focus** — Tailored for medical virtual assistance
- 🎨 **Dark & Gold Theme** — Premium, professional aesthetic
- 🖱️ **Interactive Animations** — 3D tilt effects, flip cards, particle background
- 📱 **Fully Responsive** — Optimized for all devices
- 📝 **Flip Card Testimonials** — Interactive client reviews
- 🏷️ **Case Study Modals** — Detailed project breakdowns
- 🔗 **Social Media Integration** — LinkedIn, Instagram, X, YouTube, TikTok, GitHub
- 📧 **Contact Form** — Integrated with Formspree
- 📄 **Resume Download** — Direct PDF download
- 🎯 **HIPAA Certified Badge** — Trust signal for healthcare clients
